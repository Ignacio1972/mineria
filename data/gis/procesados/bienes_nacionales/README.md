# Bienes Nacionales Protegidos

## Descripción
Terrenos fiscales con protección ambiental administrados por el Ministerio de Bienes Nacionales.

## Tipos incluidos
- Bienes Nacionales Protegidos
- Destinaciones para conservación
- Concesiones de uso

## Fuentes oficiales

| Fuente | URL | Datos |
|--------|-----|-------|
| BBNN | https://www.bienesnacionales.cl/ | Catastro fiscal |
| IDE Chile | https://www.ide.cl/ | Capas fiscales |

## Convención de nombres
```
bienes_nacionales_{region|nacional}_{año}.geojson
```

## Campos requeridos
- `nombre`: Nombre o identificador
- `tipo`: Tipo de bien nacional
- `destino`: Uso o destino asignado
- `superficie_ha`: Superficie en hectáreas
- `decreto`: Decreto de destinación

## Sistema de referencia
- SRID: 4326 (WGS84)
- Formato: GeoJSON
