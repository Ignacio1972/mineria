# Comunas

## Descripción
División político-administrativa de Chile a nivel comunal.

## Fuentes oficiales

| Fuente | URL | Datos |
|--------|-----|-------|
| IDE Chile | https://www.ide.cl/ | División política oficial |
| SUBDERE | https://www.subdere.gov.cl/ | División administrativa |
| BCN | https://www.bcn.cl/siit/mapas_vectoriales | Mapas vectoriales |

## Convención de nombres
```
comunas_{año}.geojson
comunas_{region}_{año}.geojson
```

## Campos requeridos
- `codigo`: Código único de comuna (CUT)
- `nombre`: Nombre de la comuna
- `region_codigo`: Código de la región
- `provincia`: Nombre de la provincia

## Sistema de referencia
- SRID: 4326 (WGS84)
- Formato: GeoJSON

## Notas
Chile tiene 346 comunas (2024).
