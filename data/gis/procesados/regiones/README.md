# Regiones

## Descripción
División político-administrativa de Chile a nivel regional.

## Fuentes oficiales

| Fuente | URL | Datos |
|--------|-----|-------|
| IDE Chile | https://www.ide.cl/ | División política oficial |
| SUBDERE | https://www.subdere.gov.cl/ | División administrativa |
| BCN | https://www.bcn.cl/siit/mapas_vectoriales | Mapas vectoriales |

## Convención de nombres
```
regiones_{año}.geojson
```

## Campos requeridos
- `codigo`: Código de región (01-16)
- `nombre`: Nombre oficial de la región
