# Template Estructurado para Estudio de Impacto Ambiental (EIA)
## Proyectos Mineros en Chile

> **Documento de referencia basado en el análisis del proyecto "Ciclón Exploradora"**
> - Titular: Eco Earth Elements SpA (Minería Activa / Pampa Camarones)
> - Inversión: USD 125 millones
> - Tipo: Minería subterránea polimetálica (Cu, Zn, Ag, Pb, Au)
> - RCA Aprobada: 23 de diciembre de 2025
> - Tiempo de evaluación: ~2 años (3 ICSARA)

## PARTE I: MARCO LEGAL Y REQUISITOS

### 1.1 Cuándo se Requiere un EIA (Art. 11, Ley 19.300)

Un proyecto debe presentar EIA si genera alguno de estos efectos:

| Literal | Efecto | Aplica a Minería |
|---------|--------|------------------|
| a) | Riesgo para la salud de la población (efluentes, emisiones, residuos) | ✅ Muy común |
| b) | Efectos adversos significativos sobre recursos naturales renovables | ✅ Muy común |
| c) | Reasentamiento de comunidades o alteración de sistemas de vida | ⚠️ Caso a caso |
| d) | Localización próxima a áreas protegidas, sitios prioritarios, glaciares | ⚠️ Caso a caso |
| e) | Alteración significativa del valor paisajístico o turístico | ⚠️ Caso a caso |
| f) | Alteración de monumentos, sitios con valor antropológico, arqueológico | ✅ Frecuente |

### 1.2 Tipología de Proyecto (Art. 3, DS 40)

Para minería, el ingreso al SEIA es obligatorio cuando:
- **Letra i.1)**: Proyectos de desarrollo minero sobre **5.000 ton/mes**
- **Letra i.2)**: Plantas de beneficio de minerales
- **Letra i.3)**: Depósitos de relaves
- **Letra i.4)**: Botaderos de estériles

## PARTE II: ESTRUCTURA DEL EIA (Art. 18, DS 40)

### CAPÍTULO 1: DESCRIPCIÓN DEL PROYECTO

#### 1.1 Antecedentes Generales
- Nombre del proyecto
- Identificación del titular (razón social, RUT, representante legal)
- Objetivo del proyecto
- Tipología según Art. 3 del RSEIA
- Monto de inversión estimado
- Vida útil del proyecto

#### 1.2 Localización
- Ubicación político-administrativa (región, provincia, comuna)
- Coordenadas geográficas (UTM, Datum WGS84)
- Superficie total del proyecto
- **Cartografía obligatoria**:
  - Plano de ubicación general (escala regional)
  - Plano de emplazamiento (escala local)
  - Planos de obras e instalaciones
  - Plano de áreas de influencia

#### 1.3 Descripción de Partes, Obras y Acciones

**Para proyecto minero tipo Ciclón Exploradora:**

| Sector | Componentes |
|--------|-------------|
| **Área Mina** | Portales de acceso, galerías subterráneas, sistema de ventilación (chimeneas), sistema de drenaje interior, polvorines, botaderos de estériles |
| **Área Planta** | Planta de procesamiento (sulfuros/óxidos), chancado, molienda, flotación, espesadores, filtros |
| **Depósito de Relaves** | Muro, cubeta, sistema de impermeabilización, sistema de recirculación de aguas |
| **Infraestructura de Apoyo** | Campamento, oficinas, talleres, bodegas, planta de tratamiento de aguas servidas, relleno sanitario |
| **Obras Lineales** | Camino de acceso, línea de transmisión eléctrica (AT y MT), línea de impulsión de agua |

#### 1.4 Descripción de Fases del Proyecto

| Fase | Duración Típica | Actividades Principales |
|------|-----------------|-------------------------|
| **Construcción** | 24-36 meses | Movimiento de tierras, obras civiles, montaje de equipos |
| **Operación** | 10-20+ años | Extracción, procesamiento, manejo de residuos |
| **Cierre** | 2-5 años | Desmantelamiento, estabilización, revegetación |
| **Post-cierre** | 5-10+ años | Monitoreo, mantención de obras de cierre |

#### 1.5 Insumos y Servicios
- Agua: origen, caudal, calidad requerida
- Energía: potencia instalada, fuente
- Combustibles: tipo, volúmenes, almacenamiento
- Reactivos químicos: listado, cantidades, hojas de seguridad
- Explosivos: tipo, consumo mensual

#### 1.6 Emisiones y Residuos

| Tipo | Fuentes | Caracterización |
|------|---------|-----------------|
| **Emisiones atmosféricas** | Tronaduras, chancado, tránsito | MP10, MP2.5, gases de combustión |
| **Ruido** | Maquinaria, tronaduras | Niveles en dB(A), horarios |
| **Efluentes líquidos** | Aguas de contacto, RILES | Caudal, calidad, tratamiento |
| **Residuos sólidos** | Estériles, relaves, domésticos | Volúmenes, caracterización |
| **Residuos peligrosos** | Aceites, baterías, reactivos | Clasificación, manejo |

#### 1.7 Mano de Obra
- Personal en construcción (peak)
- Personal en operación
- Origen de la mano de obra (local/externa)
- Régimen de turnos

#### 1.8 Cronograma
- Carta Gantt detallada por fase
- Hitos críticos
- Fecha estimada de inicio de obras

### CAPÍTULO 4: PREDICCIÓN Y EVALUACIÓN DE IMPACTOS

#### 4.1 Metodología de Evaluación de Impactos

**Matriz de Leopold modificada** o metodología equivalente:

| Criterio | Descripción |
|----------|-------------|
| Carácter | Positivo / Negativo |
| Intensidad | Baja / Media / Alta |
| Extensión | Puntual / Local / Regional |
| Duración | Temporal / Permanente |
| Reversibilidad | Reversible / Irreversible |
| Probabilidad | Cierta / Probable / Poco probable |
| **Significancia** | No significativo / Significativo |

#### 4.2 Identificación de Impactos por Fase y Componente

**Matriz de Impactos (ejemplo para minería)**:

| Componente | Construcción | Operación | Cierre |
|------------|--------------|-----------|--------|
| Calidad del aire | Emisiones MP por mov. tierras | Emisiones MP por chancado, tronaduras | Emisiones por demolición |
| Ruido | Maquinaria, tránsito | Tronaduras, planta | Demolición |
| Agua superficial | Intervención cauces | Consumo, efluentes | Restauración |
| Agua subterránea | Excavaciones | Drenaje mina, infiltración | Recuperación niveles |
| Flora | Despeje vegetación | Deposición material particulado | Revegetación |
| Fauna | Pérdida hábitat, perturbación | Perturbación, mortalidad | Recolonización |
| Medio humano | Empleo, tránsito | Empleo, desarrollo local | Pérdida empleo |
| Patrimonio | Rescate arqueológico | N/A | N/A |
| Paisaje | Alteración visual | Alteración visual | Restauración |

- **Por qué genera el efecto** (justificación técnica)

### CAPÍTULO 5: PLAN DE MEDIDAS

#### 5.1 Jerarquía de Mitigación

```
1. PREVENCIÓN → Evitar el impacto
2. MINIMIZACIÓN → Reducir magnitud/duración
3. RESTAURACIÓN → Reparar el daño
4. COMPENSACIÓN → Sustituir lo perdido (última opción)
```

#### 5.2 Medidas por Componente Ambiental

##### Ejemplo: Calidad del Aire

| Impacto | Medida | Tipo | Indicador | Meta |
|---------|--------|------|-----------|------|
| Emisión MP por tránsito | Humectación de caminos | Mitigación | Frecuencia de riego | 2 veces/día en verano |
| Emisión MP por chancado | Captación y filtrado | Mitigación | Eficiencia filtros | >95% |
| Emisión MP por acopio | Cerramientos y humectación | Mitigación | MP10 en receptores | <130 µg/m³N |

#### 5.3 Formato Estándar de Medida

Para cada medida documentar:
- Nombre de la medida
- Objetivo
- Descripción detallada
- Fase de aplicación
- Lugar de aplicación
- Indicador de cumplimiento
- Meta o estándar
- Forma de control
- Oportunidad de control
- Responsable
